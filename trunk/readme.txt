== Description ==
This plugin allows you to use the services of copywriting, translating and proofreading from [TextMaster](http://www.textmaster.com/?pid=4f1db74529e1673829000009) from the admin panel of wordpress.

Ce plugin permet d'utiliser les services de r&eacute;daction, traduction et de relecture de [TextMaster](http://fr.textmaster.com/?pid=4f1db74529e1673829000009) depuis l'interface d'administration de wordpress.

== Installation ==

1. Upload `wp-textmaster/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
