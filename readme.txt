== Description ==

This plugin allows you to use the services of copywriting, translating and proofreading from [TextMaster](http://www.textmaster.com/?pid=4f1db74529e1673829000009) from the admin panel of Wordpress.

Get access to over 65,000 experts in copywriting, translation and proofreading for all your blog related needs.

- High Quality content guaranteed
- Most competitive price on the market
- Intuitive platform for specific or large tasks
- Very fast delivery time



== Installation ==



1. Upload `wp-textmaster/` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
